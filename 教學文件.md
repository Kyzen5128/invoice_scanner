## 1. 系統架構說明

這個 App 使用「前端 + 後端 + 資料庫」三層架構：

```
┌─────────────────────────────────────────┐
│            Flutter App (前端)            │
│  掃描發票 → 儲存本地 → 同步後端 → 對獎  │
└────────────────┬────────────────────────┘
                 │ HTTP (REST API)
                 │ http://10.0.2.2/invoice_scanner
                 │ ↑ Android 模擬器連到電腦的特殊 IP
                 │
┌────────────────▼────────────────────────┐
│       PHP + Apache (後端)                │
│  XAMPP 管理，開啟 Apache 就自動啟動      │
│  C:\xampp\htdocs\invoice_scanner\*.php  │
└────────────────┬────────────────────────┘
                 │ PDO (PHP 內建資料庫連線)
                 │
┌────────────────▼────────────────────────┐
│       MySQL / MariaDB (資料庫)           │
│       由 XAMPP 在背景啟動管理            │
└─────────────────────────────────────────┘
```

### 為什麼 Android 模擬器要用 `10.0.2.2`？

模擬器是一台「虛擬的手機」，有自己的網路環境：

- `localhost` = 模擬器自己（不是你的電腦）
- `10.0.2.2` = 你電腦的 localhost（由 Android 模擬器特別對應）

### 各環境的 baseUrl

| 執行環境 | baseUrl |
|---------|---------|
| Android 模擬器 | `http://10.0.2.2/invoice_scanner` |
| iOS 模擬器 | `http://localhost/invoice_scanner` |
| 實體手機（同一 WiFi）| `http://你電腦的區網IP/invoice_scanner` |

**查詢區網 IP（實體手機用）：** 在 PowerShell 執行 `ipconfig`，找 `IPv4 Address`

### 資料流程

```
掃描發票 → OCR → 進入明細頁確認
    ↓ 點擊儲存
    ├─→ 寫入本地 JSON（離線也能用）
    └─→ POST invoices.php → 同步到後端 MySQL

點擊🏆對獎
    ↓ GET check.php
    ↓ SQL JOIN 比對 invoices + winning_numbers
    └─→ 回傳中獎清單 → App 顯示結果

刪除發票
    ├─→ 刪除本地 JSON
    └─→ DELETE invoices.php?number=xxx → 同步刪後端
```

## 2. 基本概念補充

### HTTP 方法（GET / POST / DELETE）

| 方法 | 意思 | 比喻 | 本專案用途 |
|------|------|------|-----------|
| **GET** | 取得資料 | 去圖書館查書 | 查詢發票、對獎 |
| **POST** | 送出資料 | 把書交給圖書館收藏 | 儲存發票 |
| **DELETE** | 刪除資料 | 把書從館藏中報廢 | 刪除發票 |

- **GET**：資料放在 URL，例如 `?period=11502`，網址看得到
- **POST**：資料放在 body 裡，網址看不到，適合傳完整資料

### 什麼是 URL？

```
http://10.0.2.2/invoice_scanner/check.php
─┬──   ───┬───  ──────┬──────  ────┬────
 │         │           │             └ 檔案：要執行哪個 PHP
 │         │           └ 目錄：htdocs 底下的資料夾
 │         └ 主機：要連哪台電腦
 └ 通訊協定
```

### JSON — 前後端傳資料的格式

```
Flutter 送出 POST：
  body = '{"invoice_number":"WR-73786487","period":"11502","amount":100}'
            ↓
PHP 接收並解析：
  $body = json_decode(file_get_contents('php://input'), true);
            ↓
PHP 回傳：
  echo json_encode(['ok' => true, 'id' => 12]);
            ↓
Flutter 解析：
  final data = jsonDecode(res.body);  // data['id'] = 12
```

### CORS — 為什麼 PHP 要加 `Access-Control-Allow-Origin`？

Android 預設會阻擋「跨來源」請求。每個 PHP 檔案頂部都要加這幾行，告訴 Flutter「我允許你讀」：

```php
header('Content-Type: application/json; charset=utf-8');
header('Access-Control-Allow-Origin: *');
header('Access-Control-Allow-Methods: GET, POST, DELETE, OPTIONS');
header('Access-Control-Allow-Headers: Content-Type');

// 瀏覽器發 POST/DELETE 前會先送 OPTIONS 預檢，直接回 200 即可
if ($_SERVER['REQUEST_METHOD'] === 'OPTIONS') {
    http_response_code(200);
    exit;
}
```

## 4. 建立資料庫

### 4.1 資料表設計

**invoices（使用者的發票紀錄）**

| 欄位 | 型別 | 說明 |
|------|------|------|
| id | INT AUTO_INCREMENT | 主鍵，自動遞增 |
| invoice_number | VARCHAR(20) | 發票號碼（如 WR-73786487）|
| period | VARCHAR(10) | 期別（如 11502 = 民國115年1-2月）|
| amount | INT | 消費金額 |
| invoice_date | DATE | 發票日期 |
| image_path | VARCHAR(255) | 圖片路徑 |
| created_at | TIMESTAMP | 建立時間（自動填入）|

> `UNIQUE KEY uniq_invoice (invoice_number, period)` — 同號碼同期別不能重複存入

**winning_numbers（中獎號碼，由管理員維護）**

| 欄位 | 型別 | 說明 |
|------|------|------|
| id | INT AUTO_INCREMENT | 主鍵 |
| period | VARCHAR(10) | 期別 |
| prize_type | VARCHAR(20) | 獎別（特別獎、頭獎、六獎…）|
| number | VARCHAR(20) | 中獎號碼 |
| prize_amount | INT | 獎金金額 |

### 4.2 init.sql 完整內容

在 `backend/init.sql` 建立此檔案：

```sql
-- 建立發票資料庫與資料表
-- ⚠️ DROP TABLE 會清空舊資料，重新初始化時才執行

CREATE DATABASE IF NOT EXISTS invoice_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE invoice_db;

DROP TABLE IF EXISTS invoices;
DROP TABLE IF EXISTS winning_numbers;

-- 發票紀錄表
CREATE TABLE invoices (
  id             INT AUTO_INCREMENT PRIMARY KEY,
  invoice_number VARCHAR(20) NOT NULL,
  period         VARCHAR(10) NOT NULL,
  amount         INT         DEFAULT 0,
  invoice_date   DATE,
  image_path     VARCHAR(255),
  created_at     TIMESTAMP   DEFAULT CURRENT_TIMESTAMP,
  UNIQUE KEY uniq_invoice (invoice_number, period)  -- 同張發票不重複存入
);

-- 中獎號碼表（由管理員手動維護）
CREATE TABLE winning_numbers (
  id           INT AUTO_INCREMENT PRIMARY KEY,
  period       VARCHAR(10) NOT NULL,
  prize_type   VARCHAR(20) NOT NULL,
  number       VARCHAR(20) NOT NULL,
  prize_amount INT         NOT NULL,
  created_at   TIMESTAMP   DEFAULT CURRENT_TIMESTAMP
);
```

### 4.5 期別（period）編碼說明

```
民國年份 + 該期的偶數月份（補零至兩位）

2024年3月  → 民國113年 → 3-4月期  → "11304"
2026年2月  → 民國115年 → 1-2月期  → "11502"
2025年12月 → 民國114年 → 11-12月期 → "11412"
```

Flutter 計算函式：

```dart
static String periodFromDate(DateTime date) {
  final roc = date.year - 1911;
  final evenMonth = date.month % 2 == 0 ? date.month : date.month + 1;
  return '$roc${evenMonth.toString().padLeft(2, '0')}';
}
```

### 5.1 db.php

```php
<?php
// 資料庫連線設定
// XAMPP MariaDB 預設：root 帳號、無密碼、port 3306

$host   = 'localhost';
$dbname = 'invoice_db';
$user   = 'root';
$pass   = '';

try {
    // PDO：PHP 內建的資料庫連線介面，支援 prepared statement 防止 SQL injection
    $pdo = new PDO("mysql:host=$host;dbname=$dbname;charset=utf8mb4", $user, $pass);

    // 有錯誤時丟出例外，方便偵錯
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);

    // 查詢結果預設回傳關聯陣列（欄位名稱當 key）
    $pdo->setAttribute(PDO::ATTR_DEFAULT_FETCH_MODE, PDO::FETCH_ASSOC);

} catch (PDOException $e) {
    http_response_code(500);
    echo json_encode(['error' => '資料庫連線失敗: ' . $e->getMessage()]);
    exit;
}
```

### 5.2 invoices.php

```php
<?php
// 發票 API
// GET    /invoices.php             → 查詢所有發票
// POST   /invoices.php             → 新增或更新一張發票
// DELETE /invoices.php?number=xxxx → 依發票號碼刪除

header('Content-Type: application/json; charset=utf-8');
header('Access-Control-Allow-Origin: *');
header('Access-Control-Allow-Methods: GET, POST, DELETE, OPTIONS');
header('Access-Control-Allow-Headers: Content-Type');

if ($_SERVER['REQUEST_METHOD'] === 'OPTIONS') {
    http_response_code(200);
    exit;
}

require_once 'db.php';

$method = $_SERVER['REQUEST_METHOD'];

// ─── GET：查詢所有發票 ────────────────────────────────────────────
if ($method === 'GET') {
    $stmt = $pdo->query('SELECT * FROM invoices ORDER BY created_at DESC');
    $rows = $stmt->fetchAll();

    // PDO 預設回傳字串，手動轉型為數字確保 Flutter 解析正確
    $rows = array_map(function($row) {
        $row['id']     = (int)$row['id'];
        $row['amount'] = (int)$row['amount'];
        return $row;
    }, $rows);

    echo json_encode($rows);

// ─── POST：新增或更新發票 ─────────────────────────────────────────
} elseif ($method === 'POST') {

    // 從請求 body 讀取 JSON 資料
    $body           = json_decode(file_get_contents('php://input'), true);
    $invoice_number = $body['invoice_number'] ?? null;
    $period         = $body['period']         ?? null;
    $amount         = $body['amount']         ?? 0;
    $invoice_date   = $body['invoice_date']   ?? null;
    $image_path     = $body['image_path']     ?? null;

    if (!$invoice_number || !$period) {
        http_response_code(400);
        echo json_encode(['error' => 'invoice_number and period are required']);
        exit;
    }

    // 用 invoice_number + period 判斷是否已存在
    $stmt = $pdo->prepare('SELECT id FROM invoices WHERE invoice_number = ? AND period = ?');
    $stmt->execute([$invoice_number, $period]);
    $existing = $stmt->fetch();

    if ($existing) {
        // 已存在 → 更新
        $stmt = $pdo->prepare(
            'UPDATE invoices SET amount=?, invoice_date=?, image_path=? WHERE invoice_number=? AND period=?'
        );
        $stmt->execute([$amount, $invoice_date, $image_path, $invoice_number, $period]);
        echo json_encode(['ok' => true, 'id' => (int)$existing['id']]);
    } else {
        // 不存在 → 新增
        $stmt = $pdo->prepare(
            'INSERT INTO invoices (invoice_number, period, amount, invoice_date, image_path) VALUES (?,?,?,?,?)'
        );
        $stmt->execute([$invoice_number, $period, $amount, $invoice_date, $image_path]);
        echo json_encode(['ok' => true, 'id' => (int)$pdo->lastInsertId()]);
    }

// ─── DELETE：依發票號碼刪除 ───────────────────────────────────────
} elseif ($method === 'DELETE') {

    // 發票號碼從 URL query string 讀取，例如 ?number=WR-73786487
    $invoice_number = $_GET['number'] ?? null;

    if (!$invoice_number) {
        http_response_code(400);
        echo json_encode(['error' => 'number is required']);
        exit;
    }

    $stmt = $pdo->prepare('DELETE FROM invoices WHERE invoice_number = ?');
    $stmt->execute([$invoice_number]);
    echo json_encode(['ok' => true]);

} else {
    http_response_code(405);
    echo json_encode(['error' => 'Method not allowed']);
}
```

```php
} else {
```

### 5.4 check.php

```php
<?php
// 對獎 API
// GET /check.php → 比對所有發票與中獎號碼，回傳中獎清單與總獎金

header('Content-Type: application/json; charset=utf-8');
header('Access-Control-Allow-Origin: *');
header('Access-Control-Allow-Methods: GET, OPTIONS');
header('Access-Control-Allow-Headers: Content-Type');

if ($_SERVER['REQUEST_METHOD'] === 'OPTIONS') {
    http_response_code(200);
    exit;
}

require_once 'db.php';

// SQL JOIN：period 和 number 都對上才算中獎，依獎金由高到低排序
$stmt = $pdo->query('
    SELECT i.id, i.invoice_number, i.period, w.prize_type, w.prize_amount
    FROM invoices i
    JOIN winning_numbers w
      ON i.period = w.period
     AND i.invoice_number = w.number
    ORDER BY w.prize_amount DESC
');

$rows = $stmt->fetchAll();

// PDO 預設回傳字串，手動轉型為數字確保 Flutter 解析正確
$winners = array_map(function($row) {
    $row['id']           = (int)$row['id'];
    $row['prize_amount'] = (int)$row['prize_amount'];
    return $row;
}, $rows);

$total_prize = array_sum(array_column($winners, 'prize_amount'));

echo json_encode([
    'count'       => count($winners),
    'total_prize' => (int)$total_prize,
    'winners'     => $winners,
]);
```

### 步驟一：AndroidManifest.xml — 加網路權限

**位置：** `android/app/src/main/AndroidManifest.xml`

Android 預設禁止 App 發送網路請求，也禁止 HTTP（非 HTTPS）連線，兩個都要手動開放。

找到 `<manifest>` 標籤，加入 `INTERNET` 權限；
找到 `<application>` 標籤，加入 `usesCleartextTraffic="true"`：

```xml
<manifest xmlns:android="http://schemas.android.com/apk/res/android">

    <uses-permission android:name="android.permission.CAMERA" />
    <!-- ↓↓↓ 加這行：允許 App 發送網路請求 ↓↓↓ -->
    <uses-permission android:name="android.permission.INTERNET" />

    <application
        android:label="invoice_scanner"
        android:name="${applicationName}"
        android:icon="@mipmap/ic_launcher"
        android:usesCleartextTraffic="true">
        <!-- ↑ 加這個屬性：允許 HTTP 明文連線（Android 9+ 預設只允許 HTTPS）
               開發時後端用 http:// 必須加，正式上線後改用 HTTPS 可移除 -->
```


然後執行：

### 步驟三：新建 api_service.dart

在 `lib/services/` 資料夾新建 `api_service.dart`。

這個檔案**集中管理所有後端 API 呼叫**，其他檔案只需要呼叫這裡的函式，不需要知道 URL 細節。

```dart
import 'dart:convert';
import 'package:flutter/foundation.dart';
import 'package:http/http.dart' as http;

class ApiService {
  // 後端根網址
  // Android 模擬器用 10.0.2.2（等同於電腦的 localhost）
  // Apache 預設 port 80，不需要寫 :80
  static const String baseUrl = 'http://10.0.2.2/invoice_scanner';

  // 將發票日期轉換為台灣統一發票期別代碼
  // 例：2026-02-22 → 民國115年1-2月 → "11502"
  static String periodFromDate(DateTime date) {
    final roc = date.year - 1911;                                // 西元轉民國年
    final evenMonth = date.month % 2 == 0 ? date.month          // 偶數月直接用
                                          : date.month + 1;     // 奇數月 +1
    return '$roc${evenMonth.toString().padLeft(2, '0')}';        // 月份補零到兩位
  }

  // 新增或更新一張發票
  // 後端用 invoice_number + period 判斷是否已存在：存在 → 更新；不存在 → 新增
  static Future<int?> addInvoice({
    required String invoiceNumber,
    required String period,
    int amount = 0,
    String? invoiceDate,
    String? imagePath,
  }) async {
    final body = {
      'invoice_number': invoiceNumber,
      'period': period,
      'amount': amount,
      'invoice_date': invoiceDate,
      'image_path': imagePath,
    };

    debugPrint('📤 [API] POST /invoices  送出: ${jsonEncode(body)}');

    final res = await http.post(
      Uri.parse('$baseUrl/invoices.php'),
      headers: {'Content-Type': 'application/json'}, // 告訴後端資料是 JSON 格式
      body: jsonEncode(body),                         // Map 轉成 JSON 字串
    );

    debugPrint('📥 [API] POST /invoices  收到 ${res.statusCode}: ${res.body}');

    if (res.statusCode == 200) {
      return jsonDecode(res.body)['id'] as int?;
    }
    throw Exception('addInvoice failed: ${res.statusCode} ${res.body}');
  }

  // 依發票號碼刪除後端紀錄
  // 為什麼用號碼而不是 id？
  //   Flutter 本地用 UUID，後端用 MySQL 整數 id，格式不同無法對應
  //   發票號碼是兩邊共有的欄位，所以用它來刪
  static Future<void> deleteInvoiceByNumber(String invoiceNumber) async {
    // Uri.encodeComponent：將 "WR-73786487" 的 "-" 轉成 URL 安全格式
    final encoded = Uri.encodeComponent(invoiceNumber);

    debugPrint('📤 [API] DELETE /invoices.php?number=$encoded');

    final res = await http.delete(
      Uri.parse('$baseUrl/invoices.php?number=$encoded'),
    );

    debugPrint('📥 [API] DELETE /invoices.php  收到 ${res.statusCode}: ${res.body}');

    if (res.statusCode != 200) {
      throw Exception('deleteInvoiceByNumber failed: ${res.statusCode}');
    }
  }

  // 對獎：請後端比對所有發票與中獎號碼
  // 回傳格式：{ "count": 1, "total_prize": 2000000, "winners": [...] }
  static Future<Map<String, dynamic>> checkWinners() async {
    debugPrint('📤 [API] GET /check  發起對獎請求');

    final res = await http.get(Uri.parse('$baseUrl/check.php'));

    debugPrint('📥 [API] GET /check  收到 ${res.statusCode}: ${res.body}');

    if (res.statusCode == 200) {
      return Map<String, dynamic>.from(jsonDecode(res.body));
    }
    throw Exception('checkWinners failed: ${res.statusCode}');
  }
}
```

```dart
Future<void> save() async {
  // 1. 確保照片複製到永久目錄
  final savedImagePath = await _storage.copyImageToAppDirectory(
      state.imageLocalPath, state.id);
  final entityToSave = state.copyWith(imageLocalPath: savedImagePath);

  // 2. 存到本地 JSON（優先，確保離線也能用）
  await _storage.saveInvoice(entityToSave);

  // 3. 同步推送到後端（失敗不影響本地，離線也能繼續用）
  try {
    final number = entityToSave.invoiceNumber;
    final date = entityToSave.date ?? entityToSave.scannedAt;

      await ApiService.addInvoice(
        invoiceNumber: number,
        period: ApiService.periodFromDate(date),          // 日期計算期別
        amount: (entityToSave.totalAmount ?? 0).round(),  // double 轉 int
        invoiceDate: date.toIso8601String().substring(0, 10), // 只取 yyyy-MM-dd
        imagePath: entityToSave.imageLocalPath,
      );
    }
  } catch (e) {
    print('[InvoiceDetail] 後端同步失敗 (本地已儲存): $e');
  }
  // ↑↑↑ 新增結束 ↑↑↑
}
```

```dart
/// 刪除特定發票（本地 + 後端同步）
///
/// [id] — Flutter 本地的 UUID，用來刪本地 JSON
/// [invoiceNumber] — 發票號碼，用來刪後端（兩個 id 格式不同，各用各的）
Future<void> deleteInvoice(String id, {String? invoiceNumber}) async {
  try {
    // 1. 先刪本地 JSON
    await _storage.deleteInvoice(id);

    // 2. 同步刪後端（失敗只記 log，不影響本地已完成的刪除）
    if (invoiceNumber != null && invoiceNumber.isNotEmpty) {
      try {
        await ApiService.deleteInvoiceByNumber(invoiceNumber);
      } catch (e) {
        debugPrint('[InvoiceList] 後端刪除失敗 (本地已刪除): $e');
      }
    }

    // 3. 重新載入清單，觸發 UI 更新
    await loadInvoices();
  } catch (e, st) {
    state = AsyncValue.error(e, st);
  }
}
```

```dart
/// 呼叫後端對獎，顯示中獎結果
Future<void> _checkWinners(BuildContext context) async {
  // 先抓 navigator，避免 await 後 context 失效
  final navigator = Navigator.of(context, rootNavigator: true);

  // 顯示載入轉圈
  showDialog(
    context: context,
    barrierDismissible: false,
    useRootNavigator: true,
    builder: (_) => const Center(child: CircularProgressIndicator()),
  );

  try {
    final result = await ApiService.checkWinners();

    // 等一個 frame 讓 dialog 動畫完成，避免 _debugLocked 錯誤
    await Future.delayed(Duration.zero);
    if (navigator.canPop()) navigator.pop(); // 關閉轉圈
    if (!context.mounted) return;

    final int count = (result['count'] as int?) ?? 0;
    final int total = (result['total_prize'] as int?) ?? 0;
    final List winners = (result['winners'] as List?) ?? [];

    showDialog(
      context: context,
      builder: (c) => AlertDialog(
        shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(20)),
        title: Row(
          children: [
            Icon(
              count > 0 ? Icons.emoji_events_rounded : Icons.sentiment_neutral_rounded,
              color: count > 0 ? Colors.amber : Colors.grey,
            ),
            const SizedBox(width: 8),
            Text(count > 0 ? '恭喜中獎!' : '本次未中獎'),
          ],
        ),
        content: Column(
          mainAxisSize: MainAxisSize.min,
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            Text('中獎張數：$count 張'),
            Text('總獎金：\$$total'),
            if (winners.isNotEmpty) ...[
              const Divider(height: 24),
              const Text('明細：', style: TextStyle(fontWeight: FontWeight.bold)),
              const SizedBox(height: 8),
              ...winners.map((w) {
                final m = Map<String, dynamic>.from(w as Map);
                return Padding(
                  padding: const EdgeInsets.symmetric(vertical: 4),
                  child: Text(
                    '• ${m['invoice_number']}  ${m['prize_type']}  \$${m['prize_amount']}',
                  ),
                );
              }),
            ],
          ],
        ),
        actions: [
          TextButton(
            onPressed: () => Navigator.pop(c),
            child: const Text('關閉'),
          ),
        ],
      ),
    );
  } catch (e) {
    await Future.delayed(Duration.zero);
    if (navigator.canPop()) navigator.pop();
    if (!context.mounted) return;

    showDialog(
      context: context,
      builder: (c) => AlertDialog(

        content: Text('無法連線到後端伺服器：\n$e'),

        ],
      ),
    );
  }
```

## 8. 常見問題

### Q1：App 一直轉圈，後端沒有回應

依序確認：
1. XAMPP → Apache 是否為綠色？
2. XAMPP → MySQL 是否為綠色？
3. `AndroidManifest.xml` 有沒有加 `INTERNET` 權限和 `usesCleartextTraffic="true"`？
4. Flutter `baseUrl` 是否是 `http://10.0.2.2/invoice_scanner`？

### Q2：對獎回傳 count: 0

最常見原因：**期別（period）不符合**。

```powershell
# 查發票的 period
& "C:\xampp\mysql\bin\mysql.exe" -u root -e "SELECT invoice_number, period FROM invoice_db.invoices;"

# 查中獎號碼的 period
& "C:\xampp\mysql\bin\mysql.exe" -u root -e "SELECT period, number FROM invoice_db.winning_numbers;"
```

兩邊的 `period` 必須完全一樣才能對到。

### Q3：PHP 頁面顯示程式碼，沒有執行

原因：Junction 沒建立，或 Apache 沒啟動。

```powershell
# 確認 Junction 存在
Get-Item "C:\xampp\htdocs\invoice_scanner"

# 重新建立（路徑換成你自己的）
cmd /c mklink /J "C:\xampp\htdocs\invoice_scanner" "你的專案路徑\backend"
```

### Q4：`Invoke-RestMethod` 測試正常，但 App 還是連不到

確認 `AndroidManifest.xml` 的修改有沒有存檔，然後**完整重新 build**：

```bash
flutter clean
flutter run
```
